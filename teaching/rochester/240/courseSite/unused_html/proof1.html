Set 0 < A=alpha < 1 and let a,b > 0. Then:
a^A b^(1-A) = A a^A b^(1-A) + (1-A) a^A b^(1-A)
= Aa [ (b/a)^(1-A) ] + (1-A)b [ (a/b)^A ].
Define a function f(x)=Aa x^(1-A) + (1-A)b (1/x)^A
Then what we have so far is:
a^A b^(1-A) = f(b/a).
Let us study f, first notice that it is a differentiable function on (0,infty), with derivative:
f'(x) = Aa(1-A) x^(-A) - (1-A)bA (1/x)^(A+1).
One then checks that for x in (0, infty):
f'(x)=0 when x = b/a and
f'(x)<0 when x < b/a and
f'(x)>0 when x > b/a
Thus we conclude from calculus that f has a global (strict) minimum on (0, infty) at x=b/a.
Then since a^A b^(1-A)=f(b/a) we see that
[a^A b^(1-A)=f(b/a)] <= [f(1)=Aa + (1-A)b] with equality only if b/a=1.
That concludes the proof.
